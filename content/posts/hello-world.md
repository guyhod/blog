first post. testing the pipeline.

the workflow is simple:

1. write a markdown file in obsidian
2. drop it in `content/posts/`
3. `git push`
4. github actions builds it with hugo
5. it shows up on www.guyhod.com a minute later

a few things this theme does well out of the box:

- code blocks with syntax highlighting
- inline `code` looks like terminal output
- lists, blockquotes, headings — all monospaced
- ascii art renders cleanly inside fenced code blocks
